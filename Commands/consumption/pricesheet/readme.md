# [Group] _consumption pricesheet_

Inspect the price sheet of an Azure subscription within a billing period.

## Subgroups

- [default](/Commands/consumption/pricesheet/default/readme.md)
: Manage Default

## Commands

- [show](/Commands/consumption/pricesheet/_show.md)
: Get the price sheet for a subscription. Price sheet is available via this API only for May 1, 2014 or later.
