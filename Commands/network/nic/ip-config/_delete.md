# [Command] _network nic ip-config delete_

Delete an IP configuration.

An NIC must have at least one IP configuration.

#### examples

- Delete an IP configuration.
    ```bash
        network nic ip-config delete -g MyResourceGroup -n MyIpConfig --nic-name MyNic
    ```
