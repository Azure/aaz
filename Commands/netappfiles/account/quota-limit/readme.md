# [Group] _netappfiles account quota-limit_

Manage Quota Limit

## Commands

- [list](/Commands/netappfiles/account/quota-limit/_list.md)
: List a list of quota limits for all quotas that are under account. Currently PoolsPerAccount is the only one.

- [show](/Commands/netappfiles/account/quota-limit/_show.md)
: Get the default, current and usages account quota limit
