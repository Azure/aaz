# [Group] _monitor dashboard_

Manage Dashboard with Grafana resources

## Commands

- [create](/Commands/monitor/dashboard/_create.md)
: Create a dashboard for grafana resource. This API is idempotent, so user can either create a new dashboard or update an existing dashboard.

- [delete](/Commands/monitor/dashboard/_delete.md)
: Delete a dashboard for Grafana resource.

- [list](/Commands/monitor/dashboard/_list.md)
: List all resources of dashboards under the specified resource group.

- [show](/Commands/monitor/dashboard/_show.md)
: Get the properties of a specific dashboard for grafana resource.

- [update](/Commands/monitor/dashboard/_update.md)
: Update a dashboard for grafana resource. This API is idempotent, so user can either create a new dashboard or update an existing dashboard.
