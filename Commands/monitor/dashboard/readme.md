# [Group] _monitor dashboard_

Manage Dashboard with Grafana resources

## Commands

- [create](/Commands/monitor/dashboard/_create.md)
: Create a Dashboard with Grafana resource. This API is idempotent, so user can either create a new dashboard or update an existing dashboard.

- [delete](/Commands/monitor/dashboard/_delete.md)
: Delete a Dashboard with Grafana resource.

- [list](/Commands/monitor/dashboard/_list.md)
: List all Dashboard with Grafana resources under the specified resource group.

- [show](/Commands/monitor/dashboard/_show.md)
: Get the properties of a specific Dashboard with Grafana resource.

- [update](/Commands/monitor/dashboard/_update.md)
: Update a Dashboard with Grafana resource. This API is idempotent, so user can either create a new dashboard or update an existing dashboard.
