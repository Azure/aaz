#### examples

- Delete a VM without a prompt for confirmation.
    ```bash
        vm delete -g MyResourceGroup -n MyVm --yes
    ```

- Delete all VMs in a resource group.
    ```bash
        vm delete --ids $(az vm list -g MyResourceGroup --query "[].id" -o tsv)
    ```
