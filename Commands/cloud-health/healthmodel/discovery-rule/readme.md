# [Group] _cloud-health healthmodel discovery-rule_

Manage discovery rules that auto-populate a health model.

## Commands

- [create](/Commands/cloud-health/healthmodel/discovery-rule/_create.md)
: Create a discovery rule.

- [delete](/Commands/cloud-health/healthmodel/discovery-rule/_delete.md)
: Delete a discovery rule.

- [list](/Commands/cloud-health/healthmodel/discovery-rule/_list.md)
: List discovery rules in a health model.

- [show](/Commands/cloud-health/healthmodel/discovery-rule/_show.md)
: Get a discovery rule.

- [update](/Commands/cloud-health/healthmodel/discovery-rule/_update.md)
: Update a discovery rule.
