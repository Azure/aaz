# [Group] _stream-analytics input_

Manage Input

## Commands

- [create](/Commands/stream-analytics/input/_create.md)
: Create an input or replaces an already existing input under an existing streaming job.

- [delete](/Commands/stream-analytics/input/_delete.md)
: Delete an input from the streaming job.

- [list](/Commands/stream-analytics/input/_list.md)
: List all of the inputs under the specified streaming job.

- [show](/Commands/stream-analytics/input/_show.md)
: Get details about the specified input.

- [test](/Commands/stream-analytics/input/_test.md)
: Tests whether an input’s datasource is reachable and usable by the Azure Stream Analytics service.

- [update](/Commands/stream-analytics/input/_update.md)
: Update an input or replaces an already existing input under an existing streaming job.
