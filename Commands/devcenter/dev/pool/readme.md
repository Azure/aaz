# [Group] _devcenter dev pool_

Manage pools.

## Commands

- [align](/Commands/devcenter/dev/pool/_align.md)
: Aligns all Dev Boxes in the pool with the current configuration.

- [list](/Commands/devcenter/dev/pool/_list.md)
: List available pools.

- [show](/Commands/devcenter/dev/pool/_show.md)
: Get a pool.
