# [Group] _documentdb mongocluster entra-user_

Manage Microsoft Entra ID users on a mongo cluster.

## Commands

- [create](/Commands/documentdb/mongocluster/entra-user/_create.md)
: Create or update a Microsoft Entra ID user on a mongo cluster.

- [delete](/Commands/documentdb/mongocluster/entra-user/_delete.md)
: Delete a Microsoft Entra ID user from a mongo cluster.

- [list](/Commands/documentdb/mongocluster/entra-user/_list.md)
: List the Microsoft Entra ID users on a mongo cluster.

- [show](/Commands/documentdb/mongocluster/entra-user/_show.md)
: Get the definition of a Microsoft Entra ID user on a mongo cluster.

- [update](/Commands/documentdb/mongocluster/entra-user/_update.md)
: Update a Microsoft Entra ID user on a mongo cluster.
