# [Command] _dns-resolver forwarding-rule show_

Gets properties of a forwarding rule in a DNS forwarding ruleset.

#### examples

- Retrieve forwarding rule in a DNS forwarding ruleset
    ```bash
        dns-resolver forwarding-rule show --ruleset-name "sampleDnsForwardingRuleset" --name "sampleForwardingRule" --resource-group "sampleResourceGroup"
    ```
